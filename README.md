## 🚀 Quick Start

```bash
# Clone repo
git clone https://github.com/Akshay-Sharma354/healthcare-voice-bot.git
cd healthcare-voice-bot

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run
streamlit run advanced_app_smart_voice.py
```

Open browser: `http://localhost:8501`

## 📊 Tech Stack

- **Frontend:** Streamlit (Python)
- **AI/NLP:** Google Gemini API + gTTS
- **Data:** Pandas, JSON
- **Language:** Python 3.14
- **Voice:** Google Cloud Text-to-Speech
